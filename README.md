# PDrone

Tool to get phone number information.
Free api.

## Installation

* `apt-get update -y`
* `apt-get upgrade -y`
* `pkg install python -y`
* `pkg install python2 -y`
* `pkg install git -y`
* `pip install lolcat`
* `pip install requests`
* `git clone https://github.com/metelity/pdrone`
* `cd $HOME`
* `ls`
* `cd ipdrone`
* `ls`
* `pdrone -s (your victim phone here)`
